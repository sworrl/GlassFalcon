🦅 GlassFalcon Installer

Get GlassFalcon onto your phone, no build toolchain required.

One-click browser flash Not yet available

A true one-click "connect your phone, click Flash" installer, the same idea as Meshtastic's or ESP Web Tools' browser flashers, needs a real WebUSB↔ADB handshake implemented in JavaScript, including generating and signing an RSA key so the phone can authorize this browser the first time (the same "Allow USB debugging?" step you'd see with a real adb client). That's genuine, security-relevant protocol work, and it hasn't been built and tested against real hardware yet, publishing it half-verified for a tool that installs drone-control software isn't a corner worth cutting. It's on the roadmap; contributions welcome (see the repo).

Automated install Works today

One command, no separate ADB download, the script fetches Google's own platform-tools for you if it's not already on your PATH, downloads the latest release, and installs it.

macOS / Linux (Terminal):

curl -fsSL https://raw.githubusercontent.com/sworrl/GlassFalcon/main/tools/install.sh | bash

Windows (PowerShell):

irm https://raw.githubusercontent.com/sworrl/GlassFalcon/main/tools/install.ps1 | iex

⚠ Piping a script straight from the internet into your shell is a real trust decision, read install.sh / install.ps1 first if you'd rather not do that blind, or clone the repo and run it locally instead: 

git clone https://github.com/sworrl/GlassFalcon.git
cd GlassFalcon
./tools/install.sh          # macOS/Linux
# or: .\tools\install.ps1    # Windows PowerShell

Manual install

  1. Enable USB debugging: Settings → About phone → tap "Build number" 7 times → Developer options → USB debugging (on).
  2. Download the latest APK: Latest release
  3. Connect your phone via USB and accept the "Allow USB debugging?" prompt on the phone screen.
  4. With adb installed: adb install GlassFalcon-latest.apk

Build it yourself

git clone https://github.com/sworrl/GlassFalcon.git
cd GlassFalcon/android
./gradlew assembleDebug
adb install app/build/outputs/apk/debug/*.apk
